What is fuge?

Fuge is a tool to help developers work with microservice style systems. Fuge provides a microservice scaffold generator to rapidly create system and service components and also an execution environment

for microservices which aims to reduce friction between processes and containers, helping you to avoid shell hell! For more information and to get started go here.
